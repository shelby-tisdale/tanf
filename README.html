This repository contains all code and data for my senior honors thesis project at Duke University, “The Diminishing Cash Safety Net: Predicting Fulfillment of TANF Goals from Unobligated Funds with Bayesian Hierarchical Modeling. This is a developing project.

Data

The document sipp_data_cleaning.qmd contains all code to create the final dataset sipp_tanf.csv included in the \data folder. Each variable in the dataset is at one of __ levels to be included in the hierarchical model. The dataset contains 511497 observations of 36 variables. Each observation is an individual in a month.

in_goal1 numeric

Indicator for whether household fits criteria to be evaluated for TANF Goal 1: “Provide assistance to needy families so that children can be cared for in their own homes or in the homes of relatives.”

in_goal1 = 1 if at least one child lives in household during any year of the survey.

in_goal2 numeric

Indicator for whether household fits criteria to be evaluated for TANF Goal 2: “End the dependence of needy parents on government benefits by promoting job preparation, work, and marriage.”

in_goal2 = 1 if household is receiving any kind of government benefits in their earliest year of participation in the study (during reference time frame).

in_goal3 numeric

Indicator for whether household fits criteria to be evaluated for TANF Goal 3: “Prevent and reduce the incidence of out-of-wedlock pregnancies”

in_goal3 = 1 if person is a woman over the age of 14 and is unmarried in reference year.

Level: Person
